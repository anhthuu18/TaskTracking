================================================================================
                         API FLOW
================================================================================

User clicks "Add" button
    ↓
AddMemberModal opens with workspace members
    ↓
User selects member and clicks "Gửi thông báo"
    ↓
Mobile app calls projectService.addMemberToProject()
    ↓
Backend validates user has admin permission
    ↓
Backend checks member not already in project
    ↓
Backend finds or creates "Member" role
    ↓
Backend adds member to project
    ↓
Backend creates in-app notification
    ↓
Backend sends email notification
    ↓
Mobile app reloads project data
    ↓
New member appears in Members tab
